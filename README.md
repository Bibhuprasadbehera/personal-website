# Bibhu Prasad Behera - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, publications, and experiences.

## Features

- **Responsive Design**: Works seamlessly on all devices and screen sizes
- **Dark/Light Mode**: Toggle between themes with persistent user preference
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Comprehensive Sections**: 
  - Hero with contact information
  - About with research interests
  - Publications with links
  - Conferences & presentations
  - Education timeline
  - Work experience
  - Machine learning projects
  - Technical skills
  - Achievements & qualifications
  - Failures & learnings
  - Contact form

## Technologies Used

- **HTML5**: Semantic markup and modern structure
- **CSS3**: Custom properties, flexbox, grid, animations, and responsive design
- **JavaScript**: Interactive features, form validation, and theme switching
- **Font Awesome**: Professional icons
- **Google Fonts**: Inter font family for clean typography
