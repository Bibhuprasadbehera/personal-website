# Bibhu Prasad Behera - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, publications, and experiences.

## Features

- **Responsive Design**: Works seamlessly on all devices and screen sizes
- **Dark/Light Mode**: Toggle between themes with persistent user preference
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Comprehensive Sections**: 
  - Hero with contact information
  - About with research interests
  - Publications with links
  - Conferences & presentations
  - Education timeline
  - Work experience
  - Machine learning projects
  - Technical skills
  - Achievements & qualifications
  - Failures & learnings
  - Contact form

## Technologies Used

- **HTML5**: Semantic markup and modern structure
- **CSS3**: Custom properties, flexbox, grid, animations, and responsive design
- **JavaScript**: Interactive features, form validation, and theme switching
- **Font Awesome**: Professional icons
- **Google Fonts**: Inter font family for clean typography

## How to View Locally

1. **Clone or download** the repository files
2. **Open** `index.html` directly in your web browser

**Or use a local server:**

```bash
python3 -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.

## Deployment

This website is optimized for GitHub Pages deployment. To deploy:

1. Push these files to a GitHub repository
2. Go to repository Settings → Pages
3. Select the main branch as source
4. Your site will be published at `https://[username].github.io/[repository-name]`

## File Structure

```
/
├── index.html          # Main HTML file
├── styles.css          # All CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Customization

To customize this portfolio for your own use:

1. Update personal information in `index.html`
2. Modify colors in the `:root` CSS variables
3. Add or remove sections as needed
4. Update the contact form action if using a backend
